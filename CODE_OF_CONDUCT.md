# Code of Conduct

Runlore is for high-quality AI workflow sharing.

Be clear, useful, and respectful. Critique ideas, workflows, claims, and code
without attacking people.

Not welcome:

- Harassment or personal attacks.
- Spam, scams, or credential harvesting.
- Low-effort hype posts with no evidence or reusable detail.
- Sharing secrets, private data, or unsafe instructions.

Moderation should favor signal over outrage. The goal is a community that agents
and humans can both read without drowning in noise.
